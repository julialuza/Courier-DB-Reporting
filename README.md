# Courier-DB-Reporting
## Project Overview

This project is a database and reporting application designed for a courier company (university project).
It demonstrates the full cycle of database design, data management, and report generation using Python and MySQL.

The project is intended for learning purposes and showcases practical skills in database modeling, SQL procedures, and Python application development.

## Technologies:
Database: MySQL
Script: Python
Documentation & Modeling: ERD diagrams, UML diagrams.

## Features
- Relational database design with tables, triggers, and stored procedures
- Python scripts for automated data management and reporting
- All database and diagram details are in: **docs/dokumentacja_projektu.pdf**

## Setup
1. Create a MySQL database.
2. Run SQL scripts from documentation **docs/dokumentacja_projektu.pdf**
3. Configure Python scripts with your database credentials.
4. Run python scripts to manage data and generate reports.
